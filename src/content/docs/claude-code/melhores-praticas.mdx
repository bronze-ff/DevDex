description: Como usar o Claude Code de forma eficiente e segura.
---

## Seja específico nos prompts

Descreva o contexto, o objetivo e as restrições. Evite pedidos vagos.

## Use CLAUDE.md

Crie um arquivo `CLAUDE.md` na raiz do projeto para definir convenções, comandos importantes e contexto persistente.

## Valide o output

Sempre revise o código gerado antes de commitar. O Claude Code é um assistente, não um substituto para revisão humana.

## Commits frequentes

Commite frequentemente para poder reverter com segurança se o Claude Code tomar uma direção errada.
