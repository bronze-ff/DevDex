## Contexto e arquivos

- Use `@arquivo.ts` para referenciar arquivos diretamente no prompt
- Use `@url` para incluir conteúdo de URLs na conversa

## Fluxo de trabalho

- Comece tarefas grandes com `/plan` para alinhar antes de codar
- Use `/compact` em conversas longas para reduzir tokens
- Prefira instruções declarativas ("crie X que faz Y") a imperativas
