## Slash Commands principais

| Comando | O que faz |
|---|---|
| `/help` | Mostra ajuda |
| `/clear` | Limpa o contexto da conversa |
| `/compact` | Compacta o histórico |
| `/review` | Revisa o código atual |
| `/commit` | Cria um commit git |
| `/pr` | Abre um pull request |

## Flags de linha de comando

```bash
claude --help          # ajuda geral
claude --model opus    # escolhe modelo
claude -p "prompt"     # modo não-interativo
```
