# DevDex

Base pessoal de conhecimento técnico — construída com [Astro](https://astro.build) + [Starlight](https://starlight.astro.build).

## Seções

- **Claude Code** — CLAUDE.md, pasta .claude, comandos, MCP, dicas, melhores práticas
- **Git** — comandos essenciais e fluxos de trabalho
- **Python** — referências e snippets
- **Docker** — imagens, containers e compose
- **JavaScript** — ES2024+, async, módulos
- **Terminal** — shell, atalhos e produtividade
- **Sites Salvos** — links e ferramentas úteis
- **Setup** — plugins, skills e CLIs para máquina nova

## Comandos

| Comando           | Ação                                        |
| :---------------- | :------------------------------------------ |
| `npm install`     | Instala dependências                        |
| `npm run dev`     | Servidor local em `localhost:4321`          |
| `npm run build`   | Build de produção para `./dist/`            |
| `npm run preview` | Preview do build antes de fazer deploy      |
